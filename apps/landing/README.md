# @genomic/landing

Standalone landing page application for the TTGen project monorepo.

Commands:
- `pnpm --filter ./apps/landing dev`
- `pnpm --filter ./apps/landing build`
- `pnpm --filter ./apps/landing start`

Notes:
- This app was extracted from `apps/web/src/app/landing`.
- The root route `/` renders landing content directly.
